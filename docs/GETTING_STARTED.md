# Ensemble — 给所有人的 AI 工作台

> **Many minds. One workspace.** — 群智，一席。

## 你有没有过这种体验

打开 Claude 写文档、切到 ChatGPT 跑代码、又跳到 Cursor 改 bug、再回头问豆包一个问题。一天下来浏览器开了八个 tab，复制粘贴的次数比写字还多。

或者，你只用一个 AI 工具，但它给的方案不够细致，让它"分头去干"它做不到。

或者，月底订阅账单一到，看着数字心头一紧——这钱到底花在哪了？

Ensemble 就是为这些痛点准备的。

## 它是什么

一个桌面应用。双击安装，开箱即用，**让多个 AI 助手在同一个窗口里互相协作**——而不是你在多个工具之间来回搬运。

## 它能干嘛

### 一个团队，几位同事

你可以同时拥有：

- 一位「研究员」用 Claude，擅长分析复杂问题
- 一位「程序员」用 Codex，本地代码执行靠它
- 一位「校对」用便宜的小模型，专门挑错

这三位 AI **可以互相递活、互相 review**。研究员写完方案直接交给程序员实现，程序员写完再让校对挑刺。整条流水线你都不用复制粘贴。

### 子助手推荐 beta

不知道下一步该让谁干？「推荐子助手」是当前辅助能力 / beta：它会根据你们刚才聊的内容，给出几个适合的角色建议，比如「代码审查员」「用户研究员」「测试工程师」。你可以按建议创建子助手或团队成员，再让它接力。

### 钱花在哪儿，清清楚楚

主界面有个小图标，点开就是花费仪表盘：

- 这周花了多少
- 哪个助手最烧钱
- 哪个模型用得最多
- 按日期、按角色，怎么拆都行

ChatGPT 订阅用户用 Codex 助手时，Ensemble 会记录 token，但**不按 API 单价重复计费**，避免把订阅型使用误算成按量 API 成本。

### 安全感拉满

AI 想要打开文件、执行命令、改你的代码？默认**先弹窗问你**。信任它就一键放行；不信任就拒绝，或切到「计划模式」——AI 只能写计划不能动手。

Codex 助手还有三档沙箱：只读、可写当前项目、完全放开。安全边界你说了算。

### 键盘党的狂欢

界面是程序员熟悉的 tmux 风格。`Ctrl+B` 后按 `%` 横向分屏、`"` 纵向分屏，9 个窗口任你切。不喜欢键盘？鼠标点点也都能用。

### 桌面执行，数据边界清楚

Ensemble 有 server / cloud infrastructure：Cloud beta 负责账号、workspace 同步、网页工作区、远程发送和部分远程设置。桌面端仍是 agent 执行端：模型调用、本地文件访问、CLI 登录态、MCP 工具和权限审批都在本机；登录网页后，桌面在线时可以向本地 agent 发送消息并查看结果。除了你主动登录 Cloud / 同步 workspace 以及 AI 模型调用所需网络请求外，对话、配置和用量审计默认保存在本机。

## 谁适合用

- **写代码的开发者**：5 分钟搭起多 AI 流水线
- **写文档的创作者**：研究、撰写、校对三位 AI 一条龙
- **做研究的同学**：多个 AI 同时从不同角度切入一个问题
- **预算敏感的人**：便宜模型干便宜活，每一分都花在刀刃上
- **被 AI 失控搞怕的人**：权限模式 + Codex 沙箱，安全边界你说了算

## 怎么开始

1. 下载安装包，双击装好（30 秒）
2. 如果要用本地 Claude 或 Codex，先在系统终端安装并登录对应 CLI：`claude` / `codex login`
3. 启动 Ensemble，打开 Providers，添加 Claude / OpenAI / Codex / 兼容供应商；本地 CLI provider 会复用本机凭证，Ensemble 不存 CLI token
4. 点击「+ team」创建团队并为每个成员选择 provider、model 和角色；也可以先新建单个 agent 开聊

普通使用不用装 Python、Node 或 Docker。只有本地 Claude / Codex provider 需要你先安装并登录对应官方 CLI；缺失 CLI 或登录态不可用时，Ensemble 会在应用内提醒。中文界面默认开启，记得你的选择，升级不丢。

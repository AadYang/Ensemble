CORE FEATURES

六个核心能力，把多模型协作放回你的桌面工作流

01

跨模型 Agent 团队

创建团队、添加成员、为每个角色独立选择 provider + model。团队 purpose 会注入成员 prompt，Claude、GPT、Codex 可通过 peer_send / peer_query 协作。

02

复用本地 CLI 授权

Claude Code 与 Codex 走本机 CLI subprocess。先在系统终端安装并登录，Ensemble 复用本机凭证、不存 token；CLI 缺失或未登录时会提醒。

03

OpenAI / Anthropic 兼容生态

除官方端点外，支持 DeepSeek、Moonshot、智谱、MiniMax、本地推理等 OpenAI / Anthropic compat 接口。一处配置，所有 agent 复用。

04

Cloud beta 远程工作区

Ensemble server / cloud infrastructure 承担账号、workspace 同步、网页工作区、远程发送和部分设置。桌面在线时，网页可向本地 agent 发送消息并查看结果。

05

Token 用量与成本统计

每一轮对话自动记账：输入 / 输出 / 缓存读 / 缓存写 token、成本拆分、本地审计明细和 CSV 导出。订阅型 Codex 不按 API 价格重复计费。

06

Skills 与 MCP 分层管理

Skills 是 SKILL.md 指令包，支持 project / ensemble / user / system 来源与 forced / disabled / auto 状态；MCP 是外部工具服务，支持 stdio / HTTP / SSE。

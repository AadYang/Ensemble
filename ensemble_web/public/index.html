CORE FEATURES

六个核心能力，重新定义你与多模型协作的方式

01

跨模型 Agent 团队

同一团队里让 Claude、GPT、Codex 同时在岗。每个角色独立选择 provider + model，通过 peer_send / peer_query 协调，把不同模型的强项编织成一条工作链。

02

复用本地 CLI 授权

已经登录过 Claude Code 或 Codex CLI?Ensemble 直接复用本机凭证,无需二次登录、不存任何 token,订阅计费方式与官方 CLI 一致。

03

OpenAI / Anthropic 兼容生态

除官方端点外,原生支持 DeepSeek、Moonshot、智谱、MiniMax、本地推理等所有 OpenAI / Anthropic compat 接口。一处配置,所有 agent 复用。

04

模型能力同台对比

同一任务并行投递给多个模型,结果并排呈现。让 Claude Opus、GPT-5、Gemini 在你真实的工作流里 PK,而不是看跑分榜。

05

Token 用量与成本统计

每一轮对话自动记账:输入 / 输出 / 缓存读 / 缓存写四项 token 分别可视化,趋势图 + 明细 + CSV 导出。按 agent / model / provider / 日期任意维度透视。

06

Skills 与 MCP 互通

完全兼容 Claude Code / Codex CLI 的 SKILL.md 格式,直接复用社区已有 skill。MCP 服务器同时支持 stdio / HTTP / SSE 三种 transport,三套 runtime 共享一份配置。

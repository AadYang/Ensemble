Ensemble 新手教程

Ensemble 是一个桌面多 Agent 工作台。你可以把 Claude、OpenAI、OpenAI Compat 和 Codex CLI agent 放在同一个 workspace 里，让它们通过 peer_send / peer_query 协作。

快速开始

  1. 下载并安装 Windows 版 Ensemble_0.0.14_x64-setup.exe。
  2. 打开左侧 Providers，添加至少一个 provider。
  3. 点击 Agent 区域的 +，选择 provider 和 model。
  4. 在聊天框输入任务，按 Enter 发送。

Agent 互相通知

peer_send 用来向另一个 agent 发送新消息，peer_query 用来读取另一个 agent 的最近上下文。

Codex CLI

  1. 先安装并登录 Codex CLI：npm i -g @openai/codex，然后运行 codex login。
  2. 在 Ensemble 的 Provider 面板添加 OpenAI -> Codex CLI。
  3. 为 Codex agent 设置 workspace 和 sandbox。
Codex CLI 仍是本机 CLI 运行态，可能影响外部 Codex CLI 会话；外部会话被打断时可用 codex resume 恢复。当前版本已通过 stdio MCP bridge 支持 Codex agent 调用 peer_send / peer_query。

MCP 和 Skills

MCP 面板支持 stdio、http、sse。Skills 使用 SKILL.md，适合沉淀可复用工作流。

常见问题

如果目标 agent 收到消息但运行失败，说明 peer_send 已送达，失败点通常在目标 provider 的 key、baseUrl 或 model。
